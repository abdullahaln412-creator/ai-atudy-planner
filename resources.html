Project method

AI-assisted development process

  1. Requirement decomposition Split the page into dashboard, task board, schedule, timer, and resource center modules.
  2. Prompt engineering Ask the AI model for page structure, responsive layout, interaction ideas, and code review suggestions.
  3. Code generation Generate semantic HTML, shared CSS, and lightweight JavaScript for local browser execution.
  4. Debugging and optimization Check mobile layout, keyboard navigation, state changes, and readable color contrast.
  5. Review and summary Compare the final pages with the original requirements and record learning outcomes.
